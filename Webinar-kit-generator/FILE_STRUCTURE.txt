WHAT YOU NEED TO DO
════════════════════════════════════════════════════════════════════════════════

STEP 1: Read Documentation (10 minutes)
  [ ] Open: README_FIGMA_FEATURE.md
  [ ] Then open: QUICK_START_FIGMA.md
  [ ] Understand the feature & setup

STEP 2: Cloudinary Account (5 minutes)
  [ ] Go to: https://cloudinary.com/users/register/free
  [ ] Create free account
  [ ] Note your Cloud Name
  [ ] Create upload preset "webinar_banners" (unsigned)

STEP 3: Configure Vercel (5 minutes)
  [ ] Go to: https://vercel.com/dashboard
  [ ] Select Webinar Kit Generator project
  [ ] Settings → Environment Variables
  [ ] Add: CLOUDINARY_CLOUD_NAME
  [ ] Add: CLOUDINARY_PRESET
  [ ] Click Save & Redeploy

STEP 4: Test Locally (10 minutes)
  [ ] Open terminal
  [ ] cd /Users/admin/Downloads/Webinar-kit-generator
  [ ] npx vercel dev
  [ ] Open http://localhost:3000
  [ ] Fill form, select "Nav banner", click button
  [ ] Verify Figma file is created

STEP 5: Deploy to Production (2 minutes)
  [ ] In terminal: vercel --prod
  [ ] Wait for deployment
  [ ] Test production URL

STEP 6: Celebrate! (∞ minutes)
  [ ] Feature is live! 🎉

WORKFLOW OVERVIEW
════════════════════════════════════════════════════════════════════════════════

User fills form (Title, Date, Time, CTA, Speaker photo)
  ↓
Clicks "Nav banner" placement
  ↓
"✨ Create in Figma" section appears
  ↓
Clicks "🚀 Create Figma Banner"
  ↓
Speaker photo uploads to Cloudinary
  ↓
API calls /api/create-figma-banner
  ↓
Figma file copied from template
  ↓
Text layers updated (title, date, time, cta)
  ↓
Speaker photo layer replaced
  ↓
Figma link returned to user
  ↓
User clicks link to open banner in Figma
  ↓
Done! Share with design agency ✅
  (Zero manual Figma editing needed)
