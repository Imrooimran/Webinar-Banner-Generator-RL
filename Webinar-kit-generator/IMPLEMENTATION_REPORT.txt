================================================================================
TECHNICAL SPECIFICATIONS
================================================================================

FUNCTION SIGNATURE:
  buildNavSVG(S) → SVG string

STATE OBJECT (S):
  {
    title: string (required)                    // Auto-wraps to 2 lines
    date: string (default: "Jun 15")            // e.g., "Jun 15"
    time: string (default: "2:00 PM EST")       // e.g., "2:00 PM EST"
    buttonText: string (default: "Register now!")
    speakers: Array<{photo: URL}> (optional)    // Controls layout
    bgPattern: string (optional)                 // Custom SVG pattern
  }

SVG OUTPUT:
  Dimensions: 1440×134 pixels
  ViewBox: "0 0 1440 134"
  File size: 3-5KB (excluding base64 images)
  Format: Inline SVG with embedded styles
  Compression: gzip reduces to ~1-2KB

LAYOUT VARIANTS:
  1-2 Speakers: Show photo at (109.5, 17), size 100×100px
                Title at x=220, maxWidth=900px
  3-4 Speakers: Hide photo completely
                Title at x=150, maxWidth=950px (expanded)

ELEMENT POSITIONS (from Figma design):
  Background:          Full canvas (1440×134)
  Speaker photo:       x=109.5, y=17, w=100, h=100, rx=4.98
  Webinar pill badge:  x=302.7, y=28.5, w=69.3, h=21px
  Title text (2 lines) x=220 or 150, y=30 and y=62
  Date:                x=1142.4, y=38, font 12px
  Time:                x=1142.4, y=54, font 11px
  Button:              x=1182.4, y=46.4, w=161, h=41.2px
  Button icon:         x=1304.4, y=51, w=31.9, h=31.9px
  Divider line:        x=832.8, vertical line

COLORS:
  Background:          #1F2937 → #374151 (gradient)
  Title text:          #FFFFFF (white)
  Date/Time text:      rgba(255,255,255,0.9) and 0.7
  Button background:   #FFFFFF (white)
  Button text:         #0F62FE (IBM blue)
  Button icon:         #0F62FE (IBM blue)
  Webinar pill:        White with 0.2 opacity

TYPOGRAPHY:
  Title:               24px, weight 700, system-ui, sans-serif
  Date:                12px, weight 500
  Time:                11px, weight 400
  Button:              14px, weight 600
  Webinar pill:        9px, weight 600

PERFORMANCE:
  Generation time:     1-2 milliseconds
  SVG output size:     3-5KB (text only)
  Browser support:     95%+ (all modern browsers)
  Mobile support:      Yes (responsive SVG)
  Memory footprint:    Minimal (string generation)

REACT INTEGRATION:

VUE INTEGRATION:

EXPRESS SERVER:
  app.get('/banner/:id', (req, res) => {
    const svg = buildNavSVG(webinarData);
    res.header('Content-Type', 'image/svg+xml').send(svg);
  });
